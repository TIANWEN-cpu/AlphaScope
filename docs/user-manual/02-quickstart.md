# 快速开始

本指南帮助你在 5 分钟内完成首次分析。

---

## 第一步：启动服务

按照[安装指南](01-installation.md)完成安装后，启动服务：

**Windows:**
双击 `scripts/start_local.bat`

**macOS / Linux:**
```bash
uvicorn backend.api.main:app --port 8000 &
cd apps/web && npm run dev &
```

浏览器打开 http://localhost:3000

---

## 第二步：配置模型

1. 点击左侧菜单 **设置**
2. 选择 **模型配置**
3. 点击 **添加 Provider**
4. 填写：
   - 名称: `DeepSeek`
   - Base URL: `https://api.deepseek.com`
   - API Key: 你的 DeepSeek API Key
5. 点击 **测试连接**
6. 看到 **连接成功** 后点击 **保存**

> 最低配置：只需一个 LLM Provider 即可开始使用。

---

## 第三步：开始分析

### 方式一：对话分析

1. 点击左侧 **新建会话**
2. 输入问题，例如：
   ```
   分析贵州茅台（600519）的投资价值
   ```
3. 系统会自动：
   - 查询行情数据
   - 搜索相关新闻
   - 调用 Agent 分析
   - 生成研究报告

### 方式二：专家团分析

1. 点击左侧 **专家团**
2. 选择 **股票合伙人团队**
3. 输入股票代码：`600519`
4. 点击 **启动分析**
5. 等待 10 位专家完成分析（约 30-60 秒）

### 方式三：K 线图分析

1. 点击左侧 **K 线分析**
2. 上传一张 K 线截图
3. 系统会：
   - 识别图表类型
   - 提取技术指标
   - 反查真实行情
   - 生成分析报告

---

## 第四步：查看报告

分析完成后：

1. 报告自动保存到 **报告中心**
2. 点击报告查看详情
3. 支持：
   - Markdown 导出
   - PDF 导出
   - 搜索历史报告
   - 按股票/日期筛选

## 核心概念

### Agent（智能体）

Agent 是具有特定人设和分析风格的 AI 分析师。系统内置 5 个核心 Agent：

| Agent | 角色 | 模型 |
|-------|------|------|
| 基本面分析师 | 财务数据、估值分析 | Claude Sonnet |
| 技术面分析师 | K 线形态、技术指标 | GPT-5.2 |
| 情绪面分析师 | 新闻情绪、市场舆情 | DeepSeek |
| 风控官 | 风险评估、仓位建议 | SenseNova |
| 散户行为分析师 | 散户情绪、资金流向 | Mimo |

### 专家团（Expert Panel）

专家团是由多个专家组成的分析团队。默认配置 **股票合伙人团队**，包含 10 位专家：

- 巴菲特派（价值投资）
- 林奇派（成长投资）
- 缠论派（技术分析）
- 宏观官（宏观策略）
- 风控官（风险控制）
- 消息面分析师（舆情分析）
- 资金面分析师（资金追踪）
- 反方分析师（逆向思维）
- 合规官（合规审查）
- 总结官（结论提炼）

### 分析模式

| 模式 | Agent 数 | 特点 | 适用场景 |
|------|----------|------|----------|
| 标准模式 | 3 | 快速、低成本 | 日常快速查看 |
| 深入模式 | 5 + Critic + 主席 | 全面、高质量 | 重要决策 |
| 自动模式 | 自动选择 | 先预筛再决定 | 平衡成本与质量 |
