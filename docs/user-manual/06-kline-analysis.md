# K 线图分析

AI-FINANCE 支持上传 K 线截图，通过视觉 AI 进行技术分析。

---

## 功能概述

K 线图分析流程：

1. **上传图片** - 支持 PNG/JPG/GIF/WebP
2. **识别图表类型** - K线/折线/柱状/表格
3. **提取技术信息** - 趋势、支撑位、压力位、形态
4. **反查真实行情** - 从数据源获取实际数据
5. **交叉验证** - 对比图像与真实数据
6. **生成报告** - 输出技术分析报告

---

## 使用方式

### 通过界面

1. 点击左侧 **K 线分析**
2. 点击 **上传图片**
3. 选择 K 线截图
4. （可选）输入股票代码和周期
5. 点击 **开始分析**
6. 等待分析完成

### 通过 API

```bash
# 上传图片
curl -X POST http://localhost:8000/api/files/upload \
  -F "file=@kline.png"

# 分析图片
curl -X POST http://localhost:8000/api/vision/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "image_path": "data/uploads/xxx_kline.png",
    "ticker": "600519",
    "period": "daily"
  }'
```

## 视觉模型配置

K 线分析使用支持视觉的 LLM 模型：

```yaml
# config/models.yaml
vision:
  provider: deepseek
  model: deepseek-chat  # 需支持视觉
```

支持视觉的模型：
- DeepSeek Chat（支持）
- Claude Sonnet 4.5（支持）
- GPT-5.2（支持）

## 常见问题

### Q: 识别不准确怎么办？

1. 检查图片是否清晰
2. 确认 K 线和指标可见
3. 提供股票代码作为提示
4. 尝试使用不同的图片

### Q: 支持美股/港股 K 线吗？

支持。系统可识别各市场的 K 线图。

### Q: 分析结果可信度如何？

视觉分析是初步判断，建议：
1. 结合真实行情数据验证
2. 参考多位专家意见
3. 不作为唯一决策依据

### Q: 图片保存在哪里？

上传的图片保存在 `data/uploads/` 目录。
