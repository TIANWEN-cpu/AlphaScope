# 报告管理与导出

AI-FINANCE 自动保存所有分析报告，支持查看、搜索、导出。

---

## 报告存储

### 存储位置

```
data/reports/
├── archive/
│   ├── 600519/
│   │   ├── 20260521-120000-贵州茅台.md
│   │   └── 20260520-150000-贵州茅台.md
│   └── 000858/
│       └── 20260521-100000-五粮液.md
└── index.json
```

### 报告元数据

每份报告包含：
- 决策（买入/卖出/观望）
- 置信度评分
- Agent 模型组合
- 兜底次数
- 审稿质量指标

## 查看报告

### 通过界面

1. 点击左侧 **报告中心**
2. 浏览报告列表
3. 点击报告查看详情

### 通过 API

```bash
# 获取报告列表
curl http://localhost:8000/api/reports

# 获取报告详情
curl http://localhost:8000/api/reports/{report_id}
```

## 搜索报告

### 按股票搜索

1. 进入 **报告中心**
2. 在搜索框输入股票代码或名称
3. 查看筛选结果

### 按日期筛选

1. 进入 **报告中心**
2. 选择日期范围
3. 查看筛选结果

### 按模式筛选

1. 进入 **报告中心**
2. 选择分析模式（标准/深入/自动）
3. 查看筛选结果

---

## 导出报告

### Markdown 导出

1. 打开报告详情
2. 点击 **导出 Markdown**
3. 保存 `.md` 文件

### PDF 导出

1. 打开报告详情
2. 点击 **导出 PDF**
3. 保存 `.pdf` 文件

### 批量导出

1. 进入 **报告中心**
2. 选择多份报告
3. 点击 **批量导出**
4. 下载 ZIP 压缩包

### 证据链

报告中的关键结论会标注证据来源：

```markdown
- 核心观点：贵州茅台估值合理
```

## 后验验证

系统会自动追踪分析决策的后续表现：

### 追踪指标

| 指标 | 说明 |
|------|------|
| 3 日收益 | 决策后 3 日涨跌幅 |
| 5 日收益 | 决策后 5 日涨跌幅 |
| 10 日收益 | 决策后 10 日涨跌幅 |
| 20 日收益 | 决策后 20 日涨跌幅 |
| 最大回撤 | 决策后最大下跌幅度 |

### 查看统计

1. 点击左侧 **回测统计**
2. 查看：
   - 买入信号准确率
   - 按 Agent 统计表现
   - 按模式统计表现

### API 端点

```bash
# 获取回测统计
curl http://localhost:8000/api/backtest/stats

# 获取 Agent 准确率
curl http://localhost:8000/api/backtest/agent-accuracy

# 获取待评估决策
curl http://localhost:8000/api/backtest/pending
```

## 备份与恢复

### 备份报告

```bash
# 备份整个 data 目录
python scripts/backup_local.py

# 或手动复制
cp -r data/reports/ backup/reports/
```

### 恢复报告

```bash
# 恢复
python scripts/restore_local.py backup/

# 或手动复制
cp -r backup/reports/ data/reports/
```

## 常见问题

### Q: 报告保存在哪里？

保存在 `data/reports/archive/` 目录，按股票代码分文件夹。

### Q: 如何删除报告？

1. 进入 **报告中心**
2. 选择报告
3. 点击 **删除**

或手动删除 `data/reports/` 下的文件。

### Q: 报告可以分享吗？

可以。导出 Markdown 或 PDF 后分享。

### Q: 历史报告会丢失吗？

不会。只要 `data/` 目录存在，报告就会保留。建议定期备份。
