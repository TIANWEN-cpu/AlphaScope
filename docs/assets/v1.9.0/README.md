## 录制建议

- 先用 **Demo 零 Key 模式**启动（引导弹窗点「先用 Demo 体验」），无需配置 Key 即可走完整路径。
- 截图统一暗色主题、隐藏个人信息 / API Key。
- `golden-path.gif` 建议用 [ScreenToGif](https://www.screentogif.com/) 或 OBS 录屏后压缩，帧率 15fps、宽度 1280 即可控制体积。
